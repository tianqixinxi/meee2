Core Features

Everything you need to stay on top of your AI coding sessions

Real-time Monitoring

Automatically detect and monitor all active AI assistant sessions. See current tools, task progress, and running time at a glance.

Permission Approval

When Claude CLI needs authorization, the Dynamic Island automatically pops up. Click Allow/Deny directly without switching to terminal.

Smart Terminal Jump

Click "Open" to automatically jump to the corresponding terminal tab. Supports Ghostty, iTerm2, Terminal.app, tmux, and cmux.

Event Sounds

Configure different system sounds for different events. Session start/end, permission requests, task completion — all customizable.

Multi-session Management

Monitor multiple AI sessions simultaneously. Sessions are sorted by activity level, so you always see what's most important.

Configurable Settings

Customize refresh intervals, sound preferences, screen position, and plugin settings through the Settings panel.

Installation

Get started in minutes

1

Download meee2

Download the latest release from GitHub and drag to Applications folder.

Download Latest
2

Configure Claude CLI Hooks

Add hook configuration to ~/.claude/settings.json

3

Start meee2

Launch meee2 and it will automatically detect and monitor your AI coding sessions.

~/.claude/settings.json 
{
  "hooks": {
    "SessionStart": [{
      "command": "/Applications/meee2.app/Contents/Resources/Bridge/meee2-bridge --source claude"
    }],
    "PermissionRequest": [{
      "command": "/Applications/meee2.app/Contents/Resources/Bridge/meee2-bridge --source claude"
    }],
    "Notification": [{
      "command": "/Applications/meee2.app/Contents/Resources/Bridge/meee2-bridge --source claude"
    }],
    "Stop": [{
      "command": "/Applications/meee2.app/Contents/Resources/Bridge/meee2-bridge --source claude"
    }]
  }
}
