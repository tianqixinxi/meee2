Core Features

Everything you need to stay on top of your AI coding sessions

Real-time Monitoring

Automatically detect and monitor all active AI assistant sessions. See current tools, task progress, and running time at a glance.

Permission Approval

When Claude CLI needs authorization, the Dynamic Island automatically pops up. Click Allow/Deny directly without switching to terminal.

Smart Terminal Jump

Click "Open" to automatically jump to the corresponding terminal tab. Supports Ghostty, iTerm2, Terminal.app, tmux, and cmux.

Event Sounds

Configure different system sounds for different events. Session start/end, permission requests, task completion — all customizable.

Plugin Tab Filter

Filter sessions by plugin source with bottom tabs. Quick switch between All, Claude, Cursor, Trae, and Aime views.

Rich Message Display

Permission requests show full toolInput details. Markdown formatting for structured information like tables and code blocks.

Configurable Settings

Customize refresh intervals, sound preferences, screen position, and plugin settings through the Settings panel.

Create Your Own Plugin

Just copy this prompt to any AI agent — they'll build it for you

Prompt for AI Agent 
Create a meee2 plugin for [YOUR_TOOL_NAME]:

Requirements:
- Plugin ID: com.meee2.plugin.[name]
- Display name: [Your Tool Name]
- Icon: [SF Symbol name]
- Theme color: [SwiftUI Color]

The plugin should:
1. Monitor [describe your tool's session/activity]
2. Return PluginSession list via getSessions()
3. Implement terminal jump via activateTerminal()

Reference:
- Template: https://github.com/tianqixinxi/meee2/tree/main/plugin-template
- Docs: https://github.com/tianqixinxi/meee2/blob/main/docs/PLUGIN_DEVELOPMENT.md
